# Budget-app
The project is about building a mobile web application where you can manage your budget: you have a list of transactions associated with a category, so that you can see how much money you spent and on what.

## Getting Started
To get a local copy up and running follow these simple example steps.

### Prerequisites
Ruby: 2.6.3 Rails: 5.2.3 Postgres: >=9.5

### Setup
Install gems with:
```javascript
bundle install
```
Setup database with:
```javascript
rails db:create
rails db:migrate
rails db:seed
```
### Usage
Start server with:
```javascript
rails server
```
Open http://localhost:3000/ in your browser.

### Run tests
Install npm with:
```javascript
npm i
```
Install rspec with:
```javascript
bundle install
```
and
```javascript
rails generate rspec:install
```

### run the test with:
```javascript
rspec spec
```
